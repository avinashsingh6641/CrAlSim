Additive cipher is monoaphabetic symmetric key cryptography, it is easiest and simlpest technique to encypt data it is also called as shift cipher ,cause we shift our message character by certain characters (Keys) ,it is not secure enough but it is easy to implement

Formula to convert message into cipher text,

C = (M+K) mod n

Formula to convert cipher into plain text,

P = (M-K) mod n

here

C => Generated cipher character

M => Message character

K => Key character

n => Total number of character used into encryption technique

Reference table for converting character value to integer
