Understanding Additive (Shift) Cipher

The Additive Cipher (historically known as the Shift Cipher or Caesar Cipher) is a classic monoalphabetic symmetric-key encryption technique. It is one of the simplest and oldest ciphers. It operates by replacing each plaintext letter with a letter found by shifting a fixed number of positions down the alphabet.

Mathematical Representation

The encryption and decryption processes can be expressed using modular arithmetic over a modulus n (usually 26, the size of the English alphabet):

Encryption Formula:
C = (M + K) % n

Decryption Formula:
P = (C - K) % n

Variables:
• C = Generated cipher character index
• M (or P) = Plaintext message character index
• K = Shift Key value (integer or key character index)
• n = Size of the alphabet (typically 26)
