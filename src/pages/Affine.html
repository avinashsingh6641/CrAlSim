Understanding Affine Cipher

The Affine Cipher is a monoalphabetic symmetric-key substitution cipher. It combines the operations of both the Multiplicative Cipher and the Additive Cipher into a single linear mathematical function: f(x) = (A * x + B) % m.

Mathematical Representation

The encryption and decryption formulas are written using modular arithmetic over alphabet size m = 26:

Encryption Formula:
C = (A * P + B) % m

Decryption Formula:
P = (A' * (C - B)) % m

Variables:
• C = Generated ciphertext character index
• P = Plaintext character index
• A = Multiplicative Key (must satisfy gcd(A, m) = 1)
• B = Additive Key
• A' = Modular Multiplicative Inverse of key A (such that (A * A') % m = 1)
• m = Total alphabet size (26)

Special Relationship Cases

The Affine Cipher is a generalization of two simpler ciphers:

  • If A = 1: The cipher acts exactly as an Additive (Shift) Cipher since C = (1 * P + B) % m => (P + B) % m.
  • If B = 0: The cipher acts exactly as a Multiplicative Cipher since C = (A * P + 0) % m => (A * P) % m.

Modulo Multiplicative Inverse (A')

The inverse key A' is needed for decryption. It exists only if gcd(A, m) = 1. In English alphabet size 26, the coprime keys are limited to 12 possible values.

Step-by-Step Example

Parameters Setup:

• Plaintext Message: "hello"

• Keys: A = 11, B = 13 (gcd(11, 26)=1)

1. Encryption (Message = "hello"):

• First letter h has index value 7.

• C = (11 * 7 + 13) % 26 = (77 + 13) % 26 = 90 % 26 = 12 (m).

• Continuing for all letters gives ciphertext: "m..."

2. Decryption:

• Find A': Since (11 * 19) % 26 = 1, modular inverse key A' = 19.

• First ciphertext letter m has index value 12.

• P = (19 * (12 - 13)) % 26 = (19 * -1) % 26 = -19 % 26 = 7 (h).
